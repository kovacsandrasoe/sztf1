Fejlesztés a parancssorból

 

Hogy elkezdhesd a munkád, szükséged van egy megfelelő szoftverre a kódok írásához!

 

Nagyon fontos megemlíteni, hogy C# nyelvű programok írásához alapesetben két dolog elegendő:

•Telepített Microsoft .NET Framework a Windows rendszerű számítógépeden

•Egy plain text szövegek írására alkalmas editor (pl: jegyzettömb/notepad, notepad++, stb.)

 

A .NET keretrendszer tartalmaz egy beépített parancssoros fordítóprogramot, amellyel a kódjaid a számítógép számára ártelmezhető köztes kódra fordulnak!

Ezt a fordítót itt érheted el: C:\Windows\Microsoft.NET\Framework\v4.0.30319 

(Ez függ a telepített .NET keretrendszertől és a rendszermeghajtód betűjelétől)

 

Tegyünk is vele egy próbát!

 

1.Indítsd el a parancssort rendszergazda jogosultsággal! (start menü -> cmd)

2.Gépeld be az alábbi parancsot: setx path "%path%;C:\Windows\Microsoft.NET\Framework\v4.0.30319\"

3.Innentől bármelyik mappában is vagy, a csc parancs beírásával fordíthatod a kódjaidat

4.Készíts egy tetszőleges mappát bárhol a számítógépen! (nálam: D:\teszt)

5.Navigálj ide a parancssorral!

6.Ehhez először írd be a meghajtó betűjelét (d:)

7.Majd add ki a cd teszt parancsot!

8.A mappában hozz létre egy first.txt fájlt az alábbi tartalommal!

 

using System;

namespace TestProject

{

    class Program

    {

        static void Main(string[] args)

        {

            int a = int.Parse(Console.ReadLine());

            int b = a * 10;

            Console.WriteLine("A beírt szám tízszerese: " + b);

            Console.ReadLine();

        }

    }

}

 

 

1.Add ki a parancssorban most a csc.exe first.txt utasítást!

2.A mappában létrejött egy first.exe fájl

3.Gratulálok! Ez az első lefordított programod!

4.Indítsd el parancssorból a first.exe utasítással!

5.Gépelj be egy számot, és megkapod a tíszeresét!

6.A program enter leütése után véget ér

 

Nos, ez láthatóan egy kényelmetlen út...

Viszont fontos megjegyezni, hogy régebben így programozott mindenki. Szerencsére napjainkban már rendelkezésünkre állnak kiváló fejlesztői környezetek (IDE: Integrated Development Environment), amelyekkel az előbbi lépéseket teljes mértékben elkerülhetjük.

Az egyik legnépszerűbb IDE a C# fejlesztők körében a Microsoft Visual Studio, amelynek a kurzuson a 2019-es, Community verzióját fogjuk használni.

Pillanatokon belül megismerkedünk vele, ám nézzük meg röviden, hogy mit biztosít nekünk egy ilyen IDE:

•Grafikus szövegszerkesztő felületet

•Beépített fordítót és debuggert (hibakeresőt)

•Kód kiegészítéseket, színezéseket

•Szerkesztési időben hiba megjelölést

•És számos más igen hasznos dolgot!
